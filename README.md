# distrowat

# Installation

`curl -fsSL https://raw.githubusercontent.com/zdk/distrowat/refs/heads/main/install.sh | sudo bash`

# Usage

## disto name

```
> distrowat name
Ubuntu
```

## codename

```
> distrowat codename
jammy
```

## version

```
> distrowat version
22.04
```

## pretty name

```
> distrowat pretty
Ubuntu 22.04.3 LTS
```
